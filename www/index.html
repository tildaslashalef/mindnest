Key Features

Mindnest combines the power of local LLMs with intelligent code analysis to deliver insightful code reviews.

AI-Powered Reviews

Uses local LLMs (Ollama or Claude API) for intelligent code analysis without sending your code to external servers.

Context-Aware Analysis

Vector embeddings help the LLM understand your codebase context for more accurate and relevant suggestions.

Git Integration

Seamlessly review staged changes, specific commits, or branch differences with built-in Git integration.

Workspace Management

Organize and track reviews across different projects with automatic workspace detection.

Intelligent Code Chunking

Smart code chunking and similarity search to provide the most relevant context for accurate reviews.

Web Sync

Seamlessly sync your CLI reviews with the web interface for team collaboration and tracking.

Quick Start Guide

Get started with Mindnest in just a few steps:

1

Install Mindnest CLI

Download and install the Mindnest CLI using our installation script.

$ curl -sSL https://mindnestapp.com/install.sh | sh
2

Install Prerequisites

For optimal performance, install Ollama for local embedding generation.

$ curl -fsSL https://ollama.com/install.sh | sh
3

Run Your First Review

Navigate to your project and run a code review.

$ cd ~/project
$ mindnest
4

Connect to Web Interface

Sync your CLI reviews with the web dashboard.

$ mindnest sync account link --token YOUR-TOKEN

Supported Features

Review Types

  • Staged changes (default)
  • Specific commits (--commit <hash>)
  • Branch difference (--branch <name>)

LLM Options

  • Local Ollama models
  • API: Claude (requires API key)

Commands

  • mindnest - Review code
  • mindnest ws info - Show workspace status
  • mindnest sync - Sync with web interface
  • mindnest --help - Show all commands
